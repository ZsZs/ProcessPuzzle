ProcessPuzzle/... is a set of libraries to easy building of an Angular application. It helps with very low level tasks like application initializing or generating tables and CRUD form run-time from entity descriptors. Utils - @processpuzzle/util Very basic functions and components
This is a collection of functions, services and components useful when building Angular applications. The package contains:
  • runtime-configuration - reads the application runtime, at app initialisation
  • app-initializer - service which helps by application initialization
  • layout-service - to help by responsive layouts
 Base Form - @processpuzzle/base-form Basis Klassen für Reactive Forms und Material Tabellen

Erleichtert die Erstellung von Forms und Tabellen mit eingebaute Unterstützung für alle übliche Funktionalität. Verbindung zum Datenquelle, State Management, automatische Daten aktualisierung, paging usw.
